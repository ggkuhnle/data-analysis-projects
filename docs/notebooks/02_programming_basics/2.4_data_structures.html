ğŸ“¦ What Are Data Structures?

When working with data â€” nutrient intakes, participant IDs, or sensory scores â€” we need containers to store, organise, and manipulate it. Different structures suit different tasks:

  • List: ordered collection of items (like a shopping list).
  • Tuple: immutable sequence (fixed recipe steps).
  • Set: unordered collection of unique elements.
  • Dictionary: keyâ€“value pairs (like nutrition labels).
  • NumPy array: fast numerical arrays (like spreadsheets optimized for math).
  • pandas DataFrame: tabular data (rows + columns, like Excel/R tables).

In this notebook, weâ€™ll explore each and see how they apply to nutrition research.

Structure Ordered? Mutable? When to Use
