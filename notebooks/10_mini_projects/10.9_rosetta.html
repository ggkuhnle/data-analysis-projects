FORTRAN 77 (fixed form) â€” ðŸŸ¡ Historical

Why include
Shows the classic numeric style still found in legacy climate/epi codes.

Idioms to notice
- Fixed columns; implicit typing unless avoided. - DO loops; FORMAT statements; simple arrays.

Snippet

30    CONTINUE
      END

Fortran (2008+ free form) â€” ðŸŸ¢ Alive

Why include
Still first-class for HPC kernels; clear array semantics and strong compilers.

Idioms to notice
- implicit none; explicit kinds; array loops or vectorised operations. - Simple, fast I/O; great for numerics.

Snippet


program nutrition_index
  implicit none
  integer, parameter :: n=5
  real :: sugar(n), sfa(n), idx(n)
  integer :: i
  do i=1,n
     read(*,*) sugar(i), sfa(i)
  end do
  idx = 0.4*sugar + 0.6*sfa
  do i=1,n
     write(*,'(F0.6)') idx(i)
  end do
end program nutrition_index

C (ISO C99) â€” ðŸŸ¢ Alive

Why include
Ubiquitous systems language; many scientific libs expose a C ABI.

Idioms to notice
- double for numeric stability. - fscanf/printf for simple text I/O.

Snippet


#include <stdio.h>
int main(void){
  double sugar, sfa, idx;
  for (int i=0;i<5;i++){
    if (scanf("%lf %lf", &sugar, &sfa)!=2) return 1;
    idx = 0.4*sugar + 0.6*sfa;
    printf("%.6f\n", idx);
  }
  return 0;
}
